# 📊 Superstore Sales Analysis

## 📌 Project Overview
This project performs **Exploratory Data Analysis (EDA)** on the Superstore dataset using **Python (Pandas, Matplotlib)** and **SQL**.
The goal is to uncover insights into **sales, profit, product performance, and regional trends**, and to visualize them for better decision-making.

---

## ⚙️ Tech Stack
- **Python** → Data cleaning, KPI calculation, visualization
- **Pandas** → Data manipulation
- **Matplotlib / Seaborn** → Charts and plots
- **SQL (MySQL)** → Query-based analysis
- **Excel / CSV** → Dataset storage and preprocessing

---

## 📂 Dataset
The dataset contains transactional records with the following key columns:
- `Order Date` → Date of purchase
- `Product Name` → Item sold
- `Category` → Product category (Furniture, Office Supplies, Technology)
- `Region` → Sales region (East, West, Central, South)
- `Sales` → Revenue generated
- `Profit` → Profit earned
